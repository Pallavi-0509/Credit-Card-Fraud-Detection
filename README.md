# 💳 Credit Card Fraud Detection

A machine learning project that detects fraudulent credit card transactions using the **LightGBM** algorithm. The model is deployed using **Streamlit** to provide an interactive interface for real-time fraud prediction.

---

## 📌 Features

- Fraud detection using LightGBM
- Interactive Streamlit web application
- Machine learning based classification
- Fast and accurate predictions
- User-friendly interface

---

## 🛠️ Technologies Used

- Python
- Pandas
- NumPy
- Scikit-Learn
- LightGBM
- Streamlit
- Joblib

## 📊 Dataset

This project uses the **Credit Card Fraud Detection Dataset** from Kaggle.

Dataset Link:
https://www.kaggle.com/datasets/mlg-ulb/creditcardfraud

> **Note:** The dataset is not included in this repository because it exceeds GitHub's file size limit. Download the dataset from Kaggle and place it inside the `data` folder before running the project.

## 🚀 Installation

Clone the repository

```bash
git clone https://github.com/Pallavi-0509/Credit-Card-Fraud-Detection.git
```

Install dependencies

```bash
pip install -r requirements.txt
```

Run the application

```bash
streamlit run app.py
```

---

## 📈 Model

- Algorithm: LightGBM
- Problem Type: Binary Classification
- Target:
  - 0 → Legitimate Transaction
  - 1 → Fraudulent Transaction
